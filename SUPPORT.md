# Support

## Getting Help

If you need help using the DevPath Bharat platform or have questions about a specific feature, there are several ways to get support:

1. **Check the FAQ**: We have a comprehensive FAQ section on our website at [devpath.org/faq](https://devpath.org/faq).
2. **Community Wiki**: Browse our [Wiki](/wiki) for in-depth documentation and guides.

## Community Support

DevPath Bharat is a community-driven project. We encourage members to help each other out!

- **GitHub Discussions**: If you have a question, idea, or want to share something with the community, head over to our [GitHub Discussions](https://github.com/devpathindcommunity-india/DevPath-Web/discussions).
- **Issues**: If you've found a bug or want to request a feature, please [open an issue](https://github.com/devpathindcommunity-india/DevPath-Web/issues). Make sure to use the provided templates.

## Discord / WhatsApp

Join our active community to chat with other developers, find mentors, and get real-time help!
- **WhatsApp Community**: [Join Here](https://chat.whatsapp.com/D2PRfQy4HYgC4XURhY2X8C)
